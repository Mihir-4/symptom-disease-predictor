# Symptom-to-Disease Prediction System

Machine Learning project for predicting diseases from symptoms using XGBoost.

## Features

- Disease prediction from symptoms
- Top-3 disease recommendations
- Confidence scoring
- Body-region symptom selection
- Streamlit web application

## Best Model

XGBoost

Accuracy: 83.51%

Top-3 Accuracy: 94.55%

## Run

pip install -r requirements.txt

streamlit run main.py
